我的Dwm配置

本文最后更新于：1 个月前

列一下我使用的Dwm配置

依赖

  • 启动器: dmenu
  • 终端: simple terminal
  • 通知守护程序: dunst
  • 浏览器: firefox
  • 截图: flameshot
  • 窗口渲染(半透明): picom
  • 锁屏: xss-lock && slock
  • 主题设置: lxappearance
  • Gtk主题:
    • Orchis
  • Qt主题:
    • 使用qt5ct和qt5-styleplugins设置Qt主题跟随Gtk2主题
  • 图标主题:
    • Papirus
  • 音乐播放器: Cmus
  • 脚本相关依赖:
    • 音量相关:
      • alsa-utils
      • pamixer
    • 亮度相关: acpilight
    • 壁纸设置: feh
    • 动态壁纸:
      • xwinwrap
      • mplayer

使用建议

autostart

启动后会自动执行~/prog/dwm/scripts/autostart.sh脚本,如果需要更改autostart脚本路径,需要到dwm.c修改.

快捷键

许多快捷键的功能通过脚本实现,建议设置一个名为DWMHOME的环境变量,确保dwm中定义的快捷键能够触发对应的脚本.

壁纸

壁纸脚本设定为2分钟自动切换,壁纸需放在~/Pictures/wallpaper目录下

登录管理器

如果要使用登录管理器,需要手动创建一个Desktop Entry
sudo vim /usr/share/xsessions/dwm.desktop
